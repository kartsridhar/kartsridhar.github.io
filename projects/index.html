MY PROJECTS

Below, is a list of most Personal and University projects I have recently worked on. The results or demos for them can be seen through the view option.

Enerjog Website

Tech Stack: VueJS, Node, ExpressJS, Heroku, SQLite

Enerjog is a fictional fitness tracker, that re-charges on movement and exercise. As part of our Web Development course, my partner and I designed and developed a single page web app using NuxtJS (a framework wrapping VueJS and Node) for the same. Furthermore, we wrote our own API to connect with a hosted database, to handle user authentication and authorisation.

Chroma: VR Puzzle Game

Tech Stack: HTC Vive, Unity, C#, C++, OpenCV, Maya 2018

Chroma is a virtual reality puzzle game incorporating drawing in 3 dimensional space as a key puzzle solving component. It does this by providing each drawing with a particular physical property based on it’s colour, which must be taken into consideration in order to solve the puzzles.

Rendering Engine

Tech Stack: GLM, SDL, C++

As part of our Computer Graphics course, my partner and I implemented a 3D Model rendering engine from scratch. This involved rasterising and raytracing the model. The renderer can apply textures, bump surfaces, smooth shaders, mirrors, metals, glass and shadows.

Dartboard Detector

Tech Stack: C++, OpenCV

As part of the Computer Vision course, my partner and I used OpenCV to implement object detection from scratch. We implemented our own Hough transform for circle and line detections. This was then combined with the Viola-Jones Object detection algorithm, trained using AdaBoost to yield accurate results. In the results, the detected boxes are in green.

3D Modelling

Tech Stack: AutoDesk Maya

Different than most technical units, I used this opportunity to explore 3D modelling from scratch. The thumbnail is a result of modelling and texturing a sports robot from scratch

Mammographic Mass Analysis

Tech Stack: Python, Tensorflow, Keras

This is a case study of comparing accuracies of different supervised classification algorithms, for the Mammographic Masses dataset provided by UC, Irvine. They include Decision Trees, Random Forest, KNN, SVM, GNB and an Artificial Neural Network.

High Performance Computing

Tech Stack: C, OpenMP, MPI, Intel Advisor

As part of the High Performance Computing Module, I was given an opportunity to work with the university super computer, BlueCrystal. The task was to serially and parallely optimise a piece of code that runs a Gaussian Blur convolution on a checkerboard image.

Pynenka: A Twitter Bot

Tech Stack: Python, Twitter API, Heroku

To experiment with Web Scraping in Python (Beautiful Soup), Pynenka is a simple twitter bot that scrapes live football scores of few games and replies them to tweets. The bot is deployed and run on a Heroku dyno.

Bristol Buddies

Tech Stack: Android Studio (Java), SpringBoot, Retrofit, Oracle VM, Nginx, MySQL

In a team of 3, we developed an app that matches transfer students with current students (their “Buddy”) who would volunteer their time to provide regular peer support and act as a reliable point of contact to help with the integration of transfer students.

Simple Spam Classifier

Tech Stack: Python, Flask, Heroku

To experiment with classification, I wrote a program that classifies text as Spam or Ham (Not Spam). I used sklearn's classification library for the same. For easier, faster and accurate predictions, I built the training model using Multinomial Naive Bayes.

Similar Movies

Tech Stack: Python

To experiment with Item Based Collaborative Filtering, I wrote a python script that finds similar movies and recommendations through correlation. For testing purposes, I used the dataset provided by MovieLens.

Kernel OS

Tech Stack: C, Assembly, Shell

As part of the Concurrent Computing coursework, I developed a kernel for Cortex-A8 emulated by QEMU. The skeleton was provided by the University.

Scotland Yard

Tech Stack: Java

The objective of the Object Oriented programming unit was to digitally implement the popular Scotland Yard board game in teams of two. The game has been well tested against 124 different edge cases making it fully playable.
