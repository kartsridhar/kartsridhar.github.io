SCOTLAND YARD

The objective of the Object Oriented programming unit was to digitally implement the popular Scotland Yard board game in teams of two. Initially, we were provided with the following:

  • The Game View and Controllers
  • The GUI
  • 124 test cases

Languages and Tools used:

  • Java
  • Intellij IDEA (IDE)

The project covered vital aspects of object oriented programming and even introduced the use of Observer Patterns in order to accomodate spectators in the game. When run, the GUI allows the user to select the number of players and assign the respective tickets. Once the game starts, Mr. X's move count and tickets are displayed on the left while the status of the Detectives on the right.
